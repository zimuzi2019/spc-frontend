本项目为本科毕设《基于微服务的生产过程中质量品控系统的设计与实现》的前端工程。

简单来说，本项目实现了上传录入表文件或者直接在线录入完成计算绘制《统计过程控制理论与实践 —— SPC、Cpk、DOE、MSA、PPM技术》
一书上提到的除缺陷成团控制图外的所有控制图。支持导出PDF或者PNG格式的计算分析报告并加以管理。也可以在单体模式和微服务模式之间
做相互切换。
